# סיפורימה — מדריך הפרויקט 🎭

ברוכה הבאה לתיקיית הפרויקט של **סיפורימה**, התיאטרון החוויתי של רימה שניידרמן.
כאן נמצא **הכל** — האתר החי, האסטרטגיה, התוכן, הסוכנים והמידע העסקי.

> 💻 **רוצה לעבוד על הפרויקט ממחשב חדש?** כל השלבים, בלי לדלג על כלום, נמצאים ב-[**SETUP.md**](SETUP.md).

---

## 🌐 האתר החי

- **כתובת:** https://sipurima.com (מאובטח 🔒, מתארח ב-Netlify)
- **הקוד של האתר:** התיקייה [`website/`](website/) — זה ורק זה מה שמתפרסם לציבור.
- **איך מפרסמים שינוי:** דוחפים ל-`main` ב-GitHub →
  ```bash
  git add -A && git commit -m "השינוי שלי" && git push origin main
  ```
  Netlify מפרסם אוטומטית תוך 1–2 דקות. אין שום שלב ידני.
- האתר מתאר את **הצגות הילדים בהשתתפות הקהל, סדנאות תיאטרון (מוכרות בגפ״ן), ימי הולדת והצגות מגבשות** — ההצגות מגיעות לכל הארץ.

## 📁 מה יש כאן?

| תיקייה | מה תמצאי שם |
|---|---|
| **`website/`** | **האתר החי** — מה שמתפרסם ל-sipurima.com (קוד, תמונות, פונטים) |

| `output/` | הפלטים היומיים — לפי תאריך |
| `scripts/` | כלי עיבוד תמונה/וידאו |

## 🤖 הסוכנים

7 סוכנים, כל אחד עם תיק מלא ב-`agents/<שם>/`:
**ceo · strategist · copywriter · workshop-designer · developer · visual-director · qa**

**כלל זהב:** רק הסוכן **CEO** מדבר ישירות איתך. כל השאר עובדים מאחורי הקלעים, וה-**QA** בודק הכל לפני שמגיע אלייך.

## ⚠️ טוב לדעת

- **הריפו פרטי** — רק החשבון `rimalasmile` ניגש אליו.
- **דחיפה ל-`main` = פרסום מיידי לאתר.** אין "טיוטה".
- **תמונות המקור** ב-`brand/` נמצאות בריפו אבל **לא** מתפרסמות לאתר (פרטי).
- **וידאו מקור כבד** לא נמצא בגיט — שמרי גיבוי בענן (פירוט ב-[SETUP.md](SETUP.md)).
