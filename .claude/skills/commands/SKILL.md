description: רשימת הפקודות הזמינות לרימה. השתמשי כשרימה רוצה להיזכר מה היא יכולה להריץ או שואלת "אילו פקודות יש?".
allowed-tools: []
---

החזירי **בדיוק** את הטקסט הבא, בלי שינויים, בלי תוספות, בלי הקדמה:

```
הפקודות שלי

  /daily        תמונת בוקר — מה חשוב היום
  /todo         מה את צריכה לעשות
  /team         מה הצוות עושה עכשיו
  /roadmap      התקדמות מול תוכנית 3 השבועות
  /decisions    מה כבר סגור
  /ask [שאלה]   שאלה חופשית להילה
  /commands     המסך הזה

התחילי את היום עם /daily. סגרי עם /todo לפני השינה.
```
