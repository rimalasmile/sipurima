## RTL ושפה מעורבת — כללי תצוגה

עברית ואנגלית באותה שורה שוברות את היישור לימין ויוצרות "קפיצות" ויזואליות.
**3 כללים — לפי הסדר:**

### 1. נתיבי קבצים תמיד בשורה נפרדת, מוזחת
לא:
```
להעלות לוגו ל-brand/logo-temp.png
```
כן:
```
להעלות לוגו
   brand/logo-temp.png
```

### 2. כל מילה באנגלית — לשורה נפרדת, מוזחת
גם מילה אחת באמצע משפט שוברת את היישור. backticks לא מספיק. הכלל פשוט: **אנגלית תמיד יורדת לשורה משלה**.

לא: הדומיין הוא sipurima.com
כן:
```
הדומיין הוא
   sipurima.com
```

### 3. אם משפט מערב יותר מדי שפות — לחתוך לשתיים
שני משפטים קצרים, כל אחד בשפה אחת, עדיפים על משפט מעורב שבור.

**למה לא סימני bidi נסתרים (RLM/LRM):** בלתי נראים, אי אפשר לדבג, מצריכים זיכרון מכל סוכן עתידי. ניוליין הוא explicit ובר-בדיקה.
